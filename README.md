# DSA4262

## DSA4262 Individual Assignment 1

This folder contains the code and data used to generate a visualization narrative on youth anxiety in Singapore.

### Repository Contents
- `DSA4262 Assignment 1.ipynb`: The Jupyter Notebook containing data processing and visualization code.
- `data/`: Folder containing raw dataset.

### How to Reproduce
1. Clone this repository.
2. Ensure you have the following libraries installed: `pip install pandas seaborn matplotlib`.
3. Open `DSA4262 Assignment 1.ipynb` and run all cells. The plots will be saved automatically as PNG files.

## DSA4262 Individual Assignment 2
This folder contains the code, data, and analytical framework used to build a binary classification model for stress detection using the Dreaddit dataset. 

### Repository Contents
- DSA4262 Assignment 2.ipynb: The primary Jupyter Notebook containing data cleaning, exploratory data analysis (EDA), feature engineering (LIWC/DAL/Social), model training, and qualitative analysis.
- DSA4262 Assignment 2.html: A static export of the completed notebook, including all generated visualizations and final results for immediate viewing.
- data/: Folder containing the dreaddit-train.csv and dreaddit-test.csv datasets.
- requirements.txt: A list of all Python dependencies required to replicate the environment.

### How to Reproduce
1. Clone this repository to your local machine.
2. Set up the environment: It is recommended to use a virtual environment.
3. Install dependencies: Run the following command to install all necessary libraries: `pip install -r requirements.txt`
4. Run the Analysis: Open DSA4262 Assignment 2.ipynb and execute all cells. The notebook will perform automated feature selection and train the final model. Visualizations will be generated and displayed inline.
